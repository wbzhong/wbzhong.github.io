JS保留两位小数的几种方法

四舍五入

参数：num为小数位数 例如：

var num1= 6.2868

num1 = num1.toFixed(2) ; //输出结果为6.29

不四舍五入

即先把小数变整数 例如：

var num1 = 6.2868;

Math.floor(num1*100)/100; //输出结果为6.28

注：这种方法有个缺点是若原来的数是整数，那么输出的结果也为整数，并没有自动补零
